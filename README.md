# Mikhael Services

Landing page mobile-first com estilo synthwave para:
- Orçamento de site
- Orçamento de edição de vídeo
- Pagamento de corrida via PIX

## Páginas
- `index.html` (hub)
- `site.html`
- `video.html`
- `corrida.html`

## Como usar
1. Abra a pasta no navegador (ou use uma extensão de servidor local).
2. Acesse `index.html`.

## Vídeo local
Para exibir o vídeo de demonstração em `video.html`, coloque seu arquivo em:

`assets/videos/apresentacao.mp4`

## Personalização rápida
No arquivo `app.js` você pode ajustar:
- E-mail
- Número e mensagem do WhatsApp
- Configuração da chave PIX

---
Projeto simples em HTML, CSS e JavaScript puro.
